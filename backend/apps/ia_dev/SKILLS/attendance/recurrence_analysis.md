# Skill: Attendance Recurrence Analysis

## Goal
Provide consistent analysis for recurring unjustified absences.

## Inputs
- Date window.
- Mode: grouped or itemized.
- Personal enrichment optional.

## Procedure
1. Resolve period.
2. Query recurrence aggregate.
3. If itemized requested, expand only recurring employees.
4. Add follow-up suggestion.
5. Emit trace and tool usage.

## Output
- Structured table payload.
- KPI summary.
- Safe natural-language response.
